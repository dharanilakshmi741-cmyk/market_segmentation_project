# Market Customer Segmentation

## Project Objective

Segment customers based on:
- Annual Income
- Spending Score
- Age
- Gender

using:
- K-Means Clustering
- PCA
- RFM Analysis

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-Learn

---

## Run Project

```bash
pip install -r requirements.txt
python main.py
```

---

## Outputs

- Elbow Method Graph
- Customer Clusters
- PCA Visualization
- Cluster Report CSV
